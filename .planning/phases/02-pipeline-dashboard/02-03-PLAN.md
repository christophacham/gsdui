<objective>
Build the wave swim lanes, plan cards (collapsed and expanded), dependency arrow visualization, agent badges, and file viewer panels. This is the core pipeline visualization -- showing what plans exist in each wave, their status, and how they depend on each other.

Purpose: Users need to see the execution structure of their pipeline -- which plans run in parallel (same wave), which are sequential, and how they connect via dependencies. Expanded cards provide drill-down into plan details.
Output: WaveContainer, WaveLane, PlanCard, PlanCardExpanded, DependencyArrows, AgentBadge, StatusIcon, MarkdownPanel, DiffPanel components plus utility functions for dependency graph traversal and SVG path generation.
